# 🛡️ CyberChallenge 2025–2026  
### Percorso di formazione in Cybersecurity  

---

## 📌 Descrizione

Questo repository raccoglie materiali, esercizi e soluzioni sviluppati durante il corso **CyberChallenge 2025–2026**, un programma di formazione avanzata dedicato alla sicurezza informatica.

L’obiettivo del corso è fornire competenze pratiche e teoriche nei principali ambiti della cybersecurity, preparando alla risoluzione di sfide realistiche (**CTF - Capture The Flag**).

---

## 🎯 Obiettivi del corso

- Comprendere i fondamenti della sicurezza informatica  
- Sviluppare capacità di analisi e problem solving  
- Imparare tecniche di attacco e difesa  
- Acquisire esperienza pratica con strumenti reali  
- Prepararsi a competizioni CTF  

---

## 📚 Argomenti trattati

### 🔐 Crittografia
- RSA (generazione chiavi, cifratura, decifratura)  
- Funzioni hash  
- Crittografia simmetrica e asimmetrica  
- Attacchi comuni (es. fattorizzazione, small exponent)  

### 🌐 Networking & Security
- Modello TCP/IP  
- NAT e configurazioni di rete  
- Analisi traffico (Wireshark)  
- Vulnerabilità di rete  

### 💻 Binary Exploitation
- Buffer overflow  
- Stack e heap  
- Reverse engineering  
- Analisi di binari  

### 🕸️ Web Security
- SQL Injection  
- XSS (Cross-Site Scripting)  
- CSRF  
- Analisi di applicazioni web  

### 🧠 Reverse Engineering
- Analisi di codice compilato  
- Uso di strumenti come Ghidra / IDA  
- Comprensione assembly base  

### 🧩 Misc / CTF Skills
- Problem solving  
- Scripting veloce (Python)  
- Automazione attacchi  

---

## 🛠️ Strumenti utilizzati

- Python (Crypto.Util.number)  
- Wireshark  
- Burp Suite  
- Ghidra  
- Netcat (nc)  
- Linux (Kali / Ubuntu)
